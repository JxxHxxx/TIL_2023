
## 1. UnknownHostException

- IP, 도메인 정보가 존재하지 않을 때 연결을 시도한 경우
- ex) 999.999.999.999, naver.coco

## 2. ConnectionException - Refused

대상 서버(IP)에서 접속은 성공했으나 연결을 거절받았을 때
![[Pasted image 20260104163223.png]](images/Pasted%20image%2020260104163223.png)

주의 : `Connection refused` 로 `Connection Timeout` 과 다른거임

다시 말해 TCP 계층에서 RST 패킷을 받았을 때 해당 예외가 발생한다.
	ex) 연결하려는 서버의 포트가 리스닝 상태가 아닐 때, 방화벽에 의해 클라이언트가 접근하지 못하는 상태일 때 


- 방화벽은 설정에 따라 다르다. 
	- 방화벽이 REJECT 다시 말해 RST 패킷을 보낼 경우에 `Connection refused` 발생
	- 방화벽이 패킷 자체를 드랍해버리면 응답이 없어서 `Connection timeout`
