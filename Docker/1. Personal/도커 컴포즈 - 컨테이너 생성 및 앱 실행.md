
프로젝트 루트 디렉토리에 `Dockerfile`, `docker-compose.yml` 생성

1. `Dockerfile`
```txt
# base image
FROM eclipse-temurin:21-jdk  
# cd app
WORKDIR /app
# 호스트머신 빌드 파일을 app,jar 로 복사 copy build/libs/hello.jar app/app.jar  
COPY build/libs/*.jar app.jar  
# 컨테이너 시작 시 최초 명령어 java -jar app.jar
ENTRYPOINT ["java", "-jar", "app.jar"] 
```

2. `docker-compose.yml`
```yaml
version: "3.8"  
  
services:  
  spring-app:  
    build: .  
    container_name: jxx-app  
    ports:  
      - "8080:8080"  
    environment:  
      SPRING_PROFILES_ACTIVE: dev
```

3. `application-dev.yml`

- 로컬 DB와 연결할 경우 아래와 같이 `url` 도메인에는 `host.docker.internal` 를 적어준다.

```yml
spring:  
  datasource:  
    url: jdbc:postgresql://host.docker.internal:5432/policy_test  
    username: {{ username }}  
    password: {{ password }}  
    driver-class-name: org.postgresql.Driver  
  
mybatis:  
  mapper-locations: classpath:/mapper/*.xml, classpath:/mapper/**/*.xml  
  configuration:  
    map-underscore-to-camel-case: true
```
